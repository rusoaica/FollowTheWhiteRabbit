Articol de importanță mică, sau discuție

Programele de calculator, cunoscute și sub numele de „produse software”, sunt o serie de instrucțiuni pe care calculatorul le poate executa. Când creați un program, trebuie să specificați instrucțiunile pe care computerul trebuie să le execute, în scopul de a realiza operațiile dorite. Procesul de definire a instrucțiunilor pe care computerul le execută se numește programare.

Instrucțiunile pe care calculatorul le execută sunt de fapt șiruri de 1 și 0 (numere binare), care reprezintă semnale electronice produse de hardware, în interiorul calculatorului. În trecut, când revoluția calculatoarelor era la început, oamenii trebuiau să înțeleagă și să programeze calculatoarele direct în aceaste șiruri de 0 și 1, numite „cod mașină”. In zilele noastre, lucrurile s-au schimbat foarte mult, programatorii scriu instrucțiuni de calculator într-o formă mai accesibilă omului, numită limbaj de programare. Un program special numit compilator transformă aceste instrucțiuni în cod mașină, care pot fi procesate de către microprocesor.

Sursă Imagine

Există mai multe limbaje de programare, fiecare cu avantaje și dezavantaje, fiecare cu diferite concepte și sintaxe; totuși, ceea ce eu voi preda este un limbaj de programare numit C# (citit ca „Si Șarp”). După părerea mea, este cel mai potrivit limbaj pentru începători, deoarece este unul dintre cel mai ușor, dar totuși robust, limbaj de programare. Sintaxa sa este, de asemenea, foarte asemănătoare cu C, C++, Java, PHP, etc, ceea ce vă va ajuta foarte mult dacă doriți vreodată să treceți la un alt limbaj (mai complex sau nu).

C# face parte dintr-o platformă de dezvoltare mai mare numită „Microsoft .NET”, și nu este singurul limbaj în care se poate dezvolta software (NET oferă de asemenea, VB.NET, F#, ASP.NET, C++ gestionat, etc). Software-ul pe care îl vom folosi pentru a scrie exercițiile și pentru a crea programele noastre se numește Visual Studio și este ceea ce programatorii numesc un „IDE” (mediu de dezvoltare integrat). Acesta oferă tot ceea ce avem nevoie pentru a proiecta, crea, scrie, compila, executa și depana folosind C#. În următoarea postare vom arunca o privire asupra instalării Visual Studio.
